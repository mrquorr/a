## Instructions

1. **[DONE]** Open: https://github.com/mrquorr/a
2. **[DONE]** Click "Code" → "Codespaces" → "Create codespace on main"
3. **[DONE]** Wait for page to load
4. Run: `bin/start <your_id> <yes|no>` in the console (replace <your_id> with your id and <yes|no> with either yes or no)

6. When you finish Round 1 run `bin/submit1 <your_id> --push`

8. Work on Round 2 (no AI)
9. When you finish Round 1 run `bin/submit2 <your_id> --push`
